# Dillon Brown's CMPA Course Portfolio
This is a portfolio dedicated to Dillon Brown's CMPA course work.
The portfolio will feature the material that I created for the course and an analysis of my actions and choices made during the courses.
Each course will have selected activities and projects for display.
The work shown will have a description of my choices and decisions to let the reader know of the process for each subject.

I chose to create a portoflio showcasing all of my work done throughout my programming course history. It serves both as a reminder of my history, and as inspiration to myself and potentially others. Allowing myself to look back on my previous work in a convenient manner lets me have easier access to view old material. This allows me, and viewers of the site, to draw inspiration from my old work to use on new projects. 

This is a simple static site built using HTML and CSS. Visual Studio Code was used to edit the code and GitHub Desktop was used to upload the code to the live site. The site is hosted using GitHub Pages. All code is written by Dillon Brown. 

You can visit the site by visiting: https://dillobrottu.github.io/dillon-cmpa-portfolio/
